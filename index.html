Featured Projects

Some of my recent work

AI Email Reply Generator

The AI-Powered Gmail Assistant is an intelligent Chrome Extension designed to streamline email communication by generating context-aware replies directly within Gmail. Built using React, Vite and Material UI for the frontend and Spring Boot for the backend, it leverages the Gemini API to craft personalized email responses in different tones — Professional, Casual or Friendly. The extension integrates seamlessly into Gmail’s interface, allowing users to generate AI-assisted replies with a single click.

Java SpringBoot React JS Vite Material UI Gemini API Chrome Extension Postman Testing

Book Finder

Book Finder is a modern and responsive web application that allows users to search for books by title, author, subject or publisher using the Open Library API. Developed with React 18, the app features a clean and interactive user interface enhanced with smooth animations, hover effects and a mobile-first design for seamless accessibility across devices. It includes functionality for users to save favorites, track search history and rate books using a 5-star review system, with all data stored persistently in the browser’s local storage.

React JS JavaScript HTML5 CSS3 Open Library API Bootstrap

To-Do App

The Task Management Application is a web-based solution designed to help users efficiently organize and monitor their daily tasks. It offers complete functionality for creating, viewing, toggling and deleting tasks. The application follows a clean, modular architecture that ensures scalability and ease of maintenance, with a clear separation between the presentation, business logic and data layers. Users can seamlessly toggle task statuses between complete and incomplete without needing to reload the page, enhancing interactivity and overall user experience.

SpringBoot JPA(Hibernate) MySQL Lombok Thymeleaf Bootstrap
